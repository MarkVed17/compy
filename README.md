Here's your features-packed note-taking app. Organize and carry anywhere all of your notes in one spot using Compy.

## Link

[compy.netlify.app](https://compy.netlify.app/)

https://user-images.githubusercontent.com/59828215/179350223-68771f1f-1aad-40f6-b6fd-98404a7bfc6a.mp4

## Features

- User Authorization
    - Sign In
    - Sign Up
    - Sign Out
- Protected/Private Routes
- Rich Text Editor
- Add Note
- Edit Note
- Sort Notes
- Filter Notes
- Archive Note
- Delete Note

## Tech Stack

**Client:** React, React Router, React Moment, Context API, [Raptor UI](https://raptor-ui.netlify.app/)

**Server:** Mockbee (Mock Backend)


## Run Locally

Clone the project

```bash
  git clone https://github.com/MarkVed17/compy.git
```

Go to the project directory

```bash
  cd compy
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
